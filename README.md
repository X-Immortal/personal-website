# 个人网站（前端静态模板）

一个简洁、现代、可直接双击打开的静态个人网站模板。无需后端、无需构建工具。

## 功能
- 响应式布局，移动端菜单
- 单页结构：关于我 / 项目 / 联系
- 平滑滚动、动态年份

## 使用
- 直接双击打开 `index.html` 预览
- 或使用任意静态服务器（可选）：
  - Python: `python -m http.server 8080`
  - Node 全局 serve: `npx serve .`

## 定制
- 修改 `index.html` 中的文案与链接
- 在 `styles.css` 中调整配色、布局
- 在 `projects` 区块添加/删除卡片

## 目录结构
```
.
├─ index.html     # 页面结构
├─ styles.css     # 样式
├─ script.js      # 交互
├─ .gitignore
└─ README.md
```
